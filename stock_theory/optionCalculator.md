# 옵션 계산기

## 1. 옵션 매매창 산출 지표 (출력)
BS(Black-Scholes) 공식 하나로 전부 산출 가능:


1. 이론가        → BS 정방향 계산
2. IV            → BS 역방향 (뉴턴-랩슨)
3. Delta         → BS를 S로 미분
4. Gamma         → Delta를 S로 미분
5. Theta         → BS를 T로 미분
6. Vega          → BS를 σ로 미분
7. 내재가치       → max(S-K, 0) 단순 뺄셈
8. 시간가치       → 프리미엄 - 내재가치
9. BEP           → 행사가 + 프리미엄
10. 손익 그래프    → 행사가별 BS 반복 계산


## 2. 외부 필요 데이터 (입력)
BS 공식만으로는 안 되고, 실시간 시세 데이터가 필요하다:

증권사 API 또는 데이터 피드에서 받아야 하는 것:

S   → KOSPI200 실시간 지수 (거래소에서 수신)
K   → 행사가 목록 (거래소에서 고정 제공)
T   → 만기일 (거래소 공시, 달력 계산)
r   → 무위험이자율 (한국은행 고시, CD91일 금리 등)
d   → 배당수익률 (추정치 사용)

옵션 호가/체결가 → IV 역산용
거래량, 미결제약정 → 단순 표시

## 3. 시스템 구현 구조 (아키텍처)
┌─────────────────────────────────────────┐
│          실시간 데이터 수신               │
│  (현물지수, 옵션호가, 거래량, 미결제약정)   │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│          BS 엔진 (핵심 계산기)            │
│                                         │
│  입력: S, K, r, d, T, σ(또는 시장가)     │
│                                         │
│  정방향: σ → 이론가                      │
│  역방향: 시장가 → IV (뉴턴-랩슨)          │
│  미분: → Delta, Gamma, Theta, Vega      │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│          화면 표시                        │
│                                         │
│  행사가 | 이론가 | 시장가 | IV | Greeks   │
│  손익그래프 | BEP | 내재가치 | 시간가치    │
└─────────────────────────────────────────┘
