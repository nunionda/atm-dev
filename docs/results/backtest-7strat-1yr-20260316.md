## 5. 업그레이드 제안

### 제안 1: multi 레짐 분류 재조정 (최우선)

현재 multi가 시장을 BEAR로 과도하게 분류 → regime_bull 대비 수익 손실. 해결:
- **INDEX_TREND 임계값 조정**: BEAR 진입 조건을 더 엄격하게 (MA200 breadth 30% → 25%)
- **NEUTRAL → BULL 전환 가속**: 5일 확인 → 3일 확인으로 단축

### 제안 2: BULL 비중을 multi에 반영

regime_bull의 가중치 구조(MR 35%, Momentum 30%)가 최적이므로:
- BULL 레짐의 `INDEX_TREND_STRATEGY_WEIGHTS`를 regime_bull과 동일하게 조정
- 현재: `{"momentum": 0.30, "smc": 0.15, "mean_reversion": 0.35, ...}`
- 이미 비슷하지만, smc 15% → 10%, defensive 15% → 10%, breakout_retest 5% → 15%로 미세 조정

### 제안 3: regime_crisis 최적화

226건 거래에 -0.5%는 비효율적:
- 인버스 헤지 85%는 실제 위기에만 효과적 — 일반 하락장에서는 평균회귀 비중 확대
- 최소 진입 임계값 상향 (score ≥ 70)으로 거래 수 감소

### 제안 4: ADX Dynamic ATR 효과 확인

MDD -2.3%로 극적 개선 → B6 ADX Dynamic ATR이 효과적임을 시사:
- 강한 추세(ADX>35)에서 1.5x 타이트 스탑 → 손실 제한
- 약한 추세(ADX<20)에서 2.5x 와이드 스탑 → 불필요한 손절 방지
