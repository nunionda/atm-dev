🧠 4-Layer Entry Scoring System

각 레이어는 독립적으로 0~25점을 산출하고, 합산 ≥ 60점이면 진입 신호가 발생합니다. 방향(LONG/SHORT)은 4개 레이어의 다수결 투표로 결정됩니다.

L1 Z-Score (25pt)

20봉 기준 Z-Score로 통계적 극단 위치 감지

z = (Close - SMA(20)) / StdDev(20) if z ≤ -2.0: # 25pt → LONG if z ≥ +2.0: # 25pt → SHORT if |z| ≥ 1.5: # 15pt if |z| ≥ 1.0: # 8pt

L2 Trend (25pt)

EMA(10/20/50) 정렬 + MA200 방향/기울기 확인

# LONG 25pt: 완전 정렬 EMA10 > EMA20 > EMA50 and Close > MA200 # LONG 18pt: 정렬 + 기울기 상승 EMA10 > EMA20 > EMA50 and slope_up # LONG 12pt: 부분 정렬 EMA10 > EMA20 and Close > MA200

L3 Momentum (25pt)

MACD 크로스 + ADX/DMI 강도 + RSI 범위 필터

# 25pt: 3중 확인 (LONG) MACD_cross_up and ADX ≥ 25 and DI+ > DI- and 30 ≤ RSI ≤ 60 # 15pt: DI + RSI 통과 # 8pt: MACD + DI만 통과

L4 Volume (25pt)

거래량 폭증 + OBV 추세 + Bollinger Band 스퀴즈

vol_spike = Vol > Avg20 × 1.5 obv_up = OBV[-1] > OBV[-3] bb_squeeze = BW < BW_avg50 × 0.8 # 25pt: spike + OBV + squeeze # 20pt: spike + OBV # 15pt: spike + 방향 일치 # 8pt: OBV + 방향 일치

🛡️ 필터 시스템

ATR 돌파 필터

현재 봉의 레인지(H-L)가 ATR × 1.2 이상이어야 신호 유효. 변동성이 너무 낮은 구간에서의 잘못된 진입 방지.

bar_range > ATR(14) × 1.2

페이크아웃 필터

저거래량 캔들(평균의 70% 미만)과 긴 위크 캔들(wick/body > 3.0)을 차단하여 노이즈 신호 제거.

if vol < avg × 0.70: block if wick/body > 3.0: block

🚪 7-Priority Exit Engine

열린 포지션에 대해 매 바마다 7개 청산 규칙을 순서대로 평가합니다. 복수 트리거 시 가장 높은 우선순위(낮은 번호) 1개만 실행됩니다.

#ID조건상세 로직
1 ES1 PnL ≤ -3% 절대 손실 한도. 어떤 상황에서도 -3% 초과 손실 방지.
2 ES_ATR_SL 동적 ATR 손절 ADX ≥ 25 (추세): Entry ± 1.5×ATR
ADX < 25 (횡보): Entry ± 2.0×ATR
3 ES_ATR_TP ATR×3 익절 Entry ± 3×ATR 도달 시 익절. R:R ≥ 2:1 보장.
4 ES_CHANDELIER 최고/최저가 - 3×ATR 22봉 최고가(롱) / 최저가(숏) 기준 3×ATR 이탈 시 청산.
5 ES3 프로그레시브 트레일링 +2% 수익에서 활성화. PnL별 타이트닝:
≥5%→0.5%, ≥3%→1.0%, else→1.5%
6 ES_CHOCH MACD 반전 MACD 히스토그램이 롱→음수 전환 또는 숏→양수 전환 시 청산.
7 ES5 최대 보유 20일 진입 시간 기준 20일 초과 시 무조건 청산.
