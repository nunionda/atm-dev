## 3. 증거금 (Margin Requirements)

### 3.1 KRX 증거금 체계

| 증거금 유형 | KOSPI200 선물 | 미니 KOSPI200 선물 |
|-------------|--------------|-------------------|
| **위탁증거금 (Initial)** | ~1,500만원 | ~300만원 |
| **유지증거금 (Maintenance)** | ~1,000만원 | ~200만원 |
| **정산증거금 (Clearing)** | SPAN 방식 산출 | SPAN 방식 산출 |

### 3.2 증거금 핵심 사항

- 증거금은 명목가치의 약 **15~23%** → 약 **4.5~6.7배 레버리지**
- KRX는 SPAN (Standard Portfolio Analysis of Risk) 시스템으로 증거금 산출
- 시장 변동성 확대 시 KRX가 증거금률 인상 가능 (임시증거금 부과)
- **위탁증거금**: 신규 포지션 진입 시 필요한 금액
- **유지증거금**: 포지션 유지를 위한 최소 금액 (하회 시 추가증거금 납부)
- 증거금 부족 시 **12:00까지** 추가증거금 납부 또는 반대매매 (브로커별 상이)
- 스프레드 거래 시 증거금 감면 적용 가능

### 3.3 레버리지 예시

```
KOSPI200 선물 1계약 매수 (KOSPI200 = 265):
  명목가치: 265 × 250,000원 = 66,250,000원
  위탁증거금: ~15,000,000원 (약 22.6%)
  레버리지: ~4.4배

  1포인트 상승 시 수익: 250,000원 → 증거금 대비 +1.67%
  1포인트 하락 시 손실: -250,000원 → 증거금 대비 -1.67%
  10포인트 하락 시 손실: -2,500,000원 → 증거금 대비 -16.7%
```

### 3.4 ES 선물과 레버리지 비교

| 항목 | KOSPI200 선물 | E-mini S&P 500 (ES) |
|------|--------------|---------------------|
| **명목가치** | ~6,625만원 (~$50,000) | ~$283,000 |
| **증거금률** | ~15~23% | ~5~5.5% |
| **레버리지** | ~4.5~6.7배 | ~18~20배 |

## 5. 가격제한폭 / 서킷브레이커 (Circuit Breaker)

### 5.1 가격제한폭 (3단계 확대)

| 단계 | 제한폭 | 적용 시간 |
|------|--------|----------|
| **1단계** | **±8%** | 전 세션 (정규 + 야간) |
| **2단계** | **±15%** | 정규거래 09:00 이후 |
| **3단계** | **±20%** | 정규거래 09:00 이후 |

- 08:45~09:00 및 야간거래 시간에는 **1단계(±8%)만 적용**
- 단계별로 확대 적용 (1단계 도달 → 2단계로 확대)

### 5.2 서킷브레이커 (CB) — 현물 연동

| 단계 | 발동 조건 | 조치 | 지속시간 |
|------|----------|------|---------|
| **1단계** | KOSPI 전일 대비 **-8%** 이상 하락 + 1분 지속 | 주식+파생 전 시장 **매매 중단** | 20분, 이후 10분 단일가 재개 |
| **2단계** | 재개 후 **-15%** 이상 하락 + 1분 지속 | 주식+파생 전 시장 **매매 중단** | 20분, 이후 10분 단일가 재개 |
| **3단계** | 재개 후 **-20%** 이상 하락 + 1분 지속 | **당일 거래 종결** | 다음 거래일까지 |

### 5.3 서킷브레이커 세부 규정

- **발동 시간 제한**: 14:50 이후에는 CB 미발동 (종가 결정에 영향 방지)
- **1일 1회**: 각 단계별 하루에 1회만 발동
- **야간거래**: 현물 가격 변동이 없으므로 CB 미적용
  - 단, 정규거래 중 CB 3단계 발동 → 당일 야간거래 미실시
- **방향**: 하락 시에만 발동 (상승 시 미발동)

### 5.4 사이드카 (SIDECAR) — 프로그램매매 일시중단

| 항목 | 내용 |
|------|------|
| **발동 조건** | 선물가격이 전일 대비 **±5% 이상 변동** 후 **1분간 지속** |
| **효과** | **프로그램 매매 호가 효력 5분간 정지** (비프로그램 매매는 계속) |
| **적용** | 정규거래 시간만 (야간 미적용) |
| **시간 제한** | 14:50 이후 미발동 |
| **1일 제한** | 같은 방향 1회만 |

### 5.5 ES 선물과 비교

| 항목 | KOSPI200 선물 | E-mini S&P 500 (ES) |
|------|--------------|---------------------|
| **1단계** | ±8% → 20분 중단 + 10분 재개 | -7% → 15분 중단 |
| **2단계** | ±15% → 20분 중단 + 10분 재개 | -13% → 15분 중단 |

| **개인 투자자** | 순포지션 **5,000계약** | 순포지션 **25,000계약** |
| **기관/법인** | 순포지션 **10,000계약** | 순포지션 **50,000계약** |
| **외국인** | 별도 제한 (외국인 한도 관리) | 별도 제한 |

### 8.2 핵심 사항

- **투기 거래**: 위의 한도 적용
- **헤지 거래**: 한도 면제 가능 (KRX 사전 승인 필요, 헤지 사유 증빙)
- **스프레드 거래**: 순포지션 계산 시 상쇄 가능
- **위반 시**: 거래 제한, 과태료, 시장조치 가능
- 미니 KOSPI200은 KOSPI200 선물과 **합산하여** 포지션 한도 관리 (미니 5계약 = 1계약 환산)

## 9. 롤오버 (Rollover)

### 9.1 분기별 롤오버 구조

1. KOSPI200 선물은 분기별로 **결제월 두 번째 목요일**에 만기
2. **롤 데이트**: 만기 약 1주일 전부터 거래량 이동 시작
3. 실무적으로 만기 **2~3 거래일 전**에 대부분 거래량이 차근월로 이동
4. 롤 이후 차근월이 "근월물"이 됨

### 9.2 2026년 롤 스케줄

| 이벤트 | 만기일 (최종거래일) | 거래량 이동 시작 | 신규 근월물 |
|--------|-------------------|----------------|-----------|
| **3월물 → 6월물** | 2026년 3월 12일 (목) | ~3월 9일 (월) | 6월물 |
| **6월물 → 9월물** | 2026년 6월 11일 (목) | ~6월 8일 (월) | 9월물 |
| **9월물 → 12월물** | 2026년 9월 10일 (목) | ~9월 7일 (월) | 12월물 |
| **12월물 → 3월물** | 2026년 12월 10일 (목) | ~12월 7일 (월) | 2027년 3월물 |

### 9.3 롤오버 실무 팁

- **최종거래일 거래 시간**: 08:45~15:20 (정규 마감보다 25분 빠름)
- 만기 당일 거래량이 급감하므로 **2~3일 전 롤오버 권장**
- **스프레드 주문** (캘린더 스프레드)으로 슬리피지 최소화
- 롤오버 시 근월-차월 간 **베이시스 차이** 확인 필수
- **동시 만기일 (쿼드러플 위칭)**: 주가지수 선물/옵션 + 개별주식 선물/옵션 동시 만기 → 극심한 변동성
- 만기일에 **최종결제가격**: KOSPI200 지수의 최종거래일 각 구성종목 최종호가의 산술평균

### 9.4 베이시스 (Basis)

```
이론 베이시스 = 선물가격 - 현물가격
           = S × (r - d) × T/365

S: KOSPI200 현물 지수
r: 무위험 이자율 (CD 91일 금리)
d: 배당수익률
T: 잔존일수
```

- **콘탱고 (Contango)**: 선물 > 현물 (정상, r > d)
- **백워데이션 (Backwardation)**: 선물 < 현물 (배당 시즌, 위기 시)
- 만기 접근 시 베이시스 → 0 수렴 (수렴 효과)

## 10. 야간거래 (Night Session) 상세

### 10.1 야간거래 특징

| 항목 | 내용 |
|------|------|
| **거래 시간** | 18:00 ~ 익일 06:00 (12시간) |
| **기초자산 연동** | 미국 시장 (S&P 500, NASDAQ) 및 유럽 시장 변동 반영 |
| **가격제한폭** | 1단계(±8%)만 적용 |
| **서킷브레이커** | 미적용 (현물 미거래 시간이므로) |
| **참여자** | 외국인 비중 높음 |
| **정산** | 다음 정규거래일 정산에 포함 |

### 10.2 야간거래 유의사항

- 정규장 CB 3단계 발동 시 → **당일 야간거래 미실시**
- 유동성이 정규장 대비 낮아 **슬리피지 확대** 가능
- 미국 경제지표 발표, FOMC 등에 따라 **급변동** 가능
- 야간거래 종가와 다음 정규장 시가 사이 **갭** 발생 가능

## 14. 트레이딩 시 주요 고려사항

### 14.1 장점

- **국내 최대 파생상품**: KRX 파생시장 거래량 1위
- **양방향 매매**: 롱/숏 동일 조건, 공매도 제한 없음
- **야간거래**: 미국장 변동에 실시간 대응 가능 (12시간)
- **프로그램매매 활발**: 차익/비차익 거래로 유동성 풍부
- **미니 계약**: 300만원 증거금으로 소액 진입 가능
- **거래세 없음**: 선물 거래에 증권거래세 미부과

### 14.2 리스크

- **레버리지**: ~4.5~6.7배 레버리지 (ES 대비 낮지만 여전히 위험)
- **갭 리스크**: 정규-야간 사이 비거래 시간 존재 (06:00~08:30, 15:45~18:00)
- **추가증거금**: 변동성 급등 시 임시증거금 인상 → 12:00까지 납부 또는 반대매매
- **만기 효과**: 동시 만기일 극심한 변동성 (쿼드러플 위칭)
- **야간 유동성**: 정규장 대비 스프레드 확대, 슬리피지 증가
- **외국인 영향**: 외국인 순매수/순매도 포지션이 시장 방향에 큰 영향

### 14.3 ATS 전략 적용 시 유의사항

| 항목 | 현물 주식 (Momentum Swing) | KOSPI200 선물 |
|------|--------------------------|---------------|
| **하드 손절** | -10% (BR-S01) | **-5%** (레버리지 감안, 더 타이트) |
| **포지션 사이징** | ATR 기반 Risk Parity | **Kelly Criterion (0.3 계수)** |
| **방향** | 롱 전용 | **양방향 (롱/숏)** |
| **최대 보유일** | 40일 (BULL) | **20일** |
| **트레일링 활성화** | 수익률 구간별 | **+2%에서 활성화** |
| **리스크/트레이드** | 1.5% of equity | **1.0% of equity** |
| **시간대 고려** | 정규장 | **정규 + 야간 (미국장 이벤트)** |
