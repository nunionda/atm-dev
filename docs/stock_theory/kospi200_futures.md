# 코스피200 선물

코스피200 선물은 유가증권시장에 상장된 주권 200 종목의 시가총액 기준으로 산출된 코스피200 지수 [산출기준시점 1990.01.03]를 기초자산으로 하는 상품입니다. 이처럼 코스피200 선물은 주가지수를 거래대상으로 하고 있어 최종결제방법으로 현금결제를 채택하고 있습니다.

## 상품명세

| **기초자산** | 코스피200지수 |

| **결제월** | 3, 6, 9, 12월 |
| **상장결제월** | 3년 이내 7개 결제월 (3, 9월 : 각 1개, 6월 : 2개, 12월 : 3개) |

| **거래시간** | **정규거래** 08:45 ~ 15:45 (최종거래일 08:45~15:20)<br>**야간거래** 18:00 ~ 06:00 |
