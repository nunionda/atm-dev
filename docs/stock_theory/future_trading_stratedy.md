# 📈 ATR(Average True Range) 기반 선물 매매 전략

선물계약 거래에서 ATR(Average True Range) 지표는 방향성이 아닌 **'시장의 변동성(Volatility)'**을 측정하는 지표로, 단순히 가격만 보고 진입하는 것이 아니라 현재 시장 상황에 맞는 유연한 진입과 청산(리스크 관리) 지점을 찾는 데 유용합니다. (기본 세팅: 보통 14일 또는 14개 봉)

---

## 1. 진입 전략 (Entry Strategy)
ATR은 가짜 돌파(False Breakout)를 걸러내고, 유의미한 변동성이 발생했을 때만 추세에 탑승하는 필터나 신호 강화 도구로 주로 활용됩니다.

* **ATR 필터링 브레이크아웃 (Breakout Strategy)**: 
  * 가격이 전일 고가, 이동평균선, 또는 저항선을 돌파할 때, `(현재 봉의 고가 - 이전 봉의 종가) > 0.5 * ATR` 조건을 만족하면 **진짜 돌파**로 간주하고 매수 진입합니다.
* **변동성 추세 추종 (Trend Following)**: 
  * 시장이 특정 방향으로 확정적인 변동성 움직임을 보일 때 진입합니다.
  * **매수**: 가격이 이동평균선 위에 있고, `현재 종가 > 이전 종가 + (1.5~2 * ATR)`일 때 강력한 상승 추세로 판단하여 매수.
  * **매도(Short)**: 가격이 이동평균선 아래에 있고, `현재 종가 < 이전 종가 - (1.5~2 * ATR)`일 때 매도.
* **ATR 밴드 활용 (Volatility Channels)**: 
  * 볼린저 밴드와 유사하게 상하단에 ATR 기반의 밴드(예: 2ATR)를 설정하고, 가격이 밴드를 벗어날 때 추세 방향으로 진입합니다.

## 2. 청산 및 리스크 관리 (Exit & Risk)
진입 후에는 변동성에 맞추어 유동적으로 익절 및 손절 라인을 설정하는 것이 핵심입니다. (시장 불안정 시 손절폭 확대, 안정 시 축소하여 잦은 스톱 방지)

* **ATR 트레일링 스톱 (Trailing Stop)**: 추세에 따라 최적의 손절가를 계속 갱신합니다.
  * **매수 포지션 손절가**: `진입가 - (1.5~2 * ATR)` (가격 상승 시 손절가도 동일 간격 상향)
  * **매도 포지션 손절가**: `진입가 + (1.5~2 * ATR)`
* **샹들리에 청산 (Chandelier Exit)**: 
  * 형성된 최고점(Long) 또는 최저점(Short)에서 `3 * ATR` 만큼 가격이 반대로 움직이면 기술적으로 즉시 포지션을 완전 종료합니다.

## 3. 백테스팅 추천 전략 (전략 심화)
단순 ATR 활용을 넘어 전략의 성능을 향상시키기 위해 다음 로직들을 추가하여 백테스팅해 보시는 것을 추천합니다.

* **동적 ATR 배수 (추세 강도에 따른 조절)**:
  * ADX 등 추세 강도 지표와 결합하여, 추세가 약한 횡보장에서는 손절폭을 넉넉하게, 추세가 강할 때는 타이트하게 가져갑니다.
  * 예시 (Pseudo-code): `multiplier = 2.0 if adx < 20 else 1.5`
* **거래량 가중 ATR (Integrated ATR)**:
  * 단순 가격 변동폭(ATR)뿐만 아니라 거래량 변화율(Volume ROC)을 가중 평균하여 돌파 신호의 신뢰도를 높입니다.
  * 예시 (Pseudo-code): `Integrated_ATR = (ATR * weight1) + (Volume_ROC * weight2)`
* **피라미딩 간격 (변동성 기반 추가 진입)**:
  * 유리한 방향으로 추세가 진행될 때 포지션을 늘리는 피라미딩(불타기) 시, 고정 가격이 아닌 '변동성(ATR)' 단위로 다단계 진입합니다.
  * 예시 (Pseudo-code): `Next_Entry = Last_Entry + (0.5 * ATR)`

## 4. 전략 요약 및 주의사항

| 구분 | 내용 |
| :--- | :--- |
| **핵심** | 가격의 '수준'이 아닌 **'움직임의 크기(변동성)'**에 따라 유연하게 대응 |
| **진입** | 횡보 후 ATR이 증가하며 주요 지지/저항 돌파 시 추세 추종 진입 |
| **청산** | 2~3배의 ATR을 활용한 동적 스톱로스(트레일링 스톱, 샹들리에 청산) |
| **장점** | 시장 상황(변동성)에 따른 합리적이고 유연한 리스크 관리 가능 |
| **단점** | 변동이 잦은 횡보장에서는 방향이 자주 바뀌어 **가짜 신호(Whipsaw)**가 발생할 위험 존재 |

> **💡 필수 팁**: ATR 지표 자체는 방향성을 알려주지 않으므로, **반드시 이동평균선(EMA)이나 추세 지표 등 방향성을 띄는 보조지표와 결합하여 사용**해야 합니다.
