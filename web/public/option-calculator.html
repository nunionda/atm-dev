🔄 IV 역산 — 시장가격과 계산법

여기에 뭘 넣나?

MTS에서 확인한 옵션의 "현재가"를 넣는다. 예시: 콜 940 현재가 12.35 → 12.35 입력 예시: 풋 930 현재가 8.50 → 8.50 입력 콜/풋 구분은 위의 토글 버튼으로 선택

IV 역산이란?

정방향 (BS 공식): IV(σ)를 넣으면 → 이론가가 나온다 C = f(S, K, T, σ, r, d) 역방향 (IV 역산): 시장가를 넣으면 → IV(σ)를 찾는다 σ = f⁻¹(S, K, T, C_market, r, d) BS 공식에는 σ에 대한 역함수가 없다 → 뉴턴-랩슨 반복법으로 수치적으로 풀어야 한다

뉴턴-랩슨법 원리

목표: BS(σ) = 시장가 를 만족하는 σ 찾기 반복 공식: σ_new = σ_old - (BS(σ_old) - 시장가) / Vega(σ_old) 1️⃣ σ = 30% 로 시작 (초기 추정값) 2️⃣ BS(30%)로 이론가 계산 3️⃣ 이론가 - 시장가 = 오차 4️⃣ 오차 / Vega = 조정량 5️⃣ σ를 조정해서 다시 2️⃣로 6️⃣ 오차 < 0.00000001 이면 수렴 완료 보통 3~5회면 수렴한다

왜 Vega로 나누나?

Vega = IV 1%p 변화 시 옵션 가격 변화량 오차가 -1.71pt이고 Vega가 0.42이면: → IV를 1.71 / 0.42 ≈ 4%p 올리면 가격이 1.71pt 오른다 → 이론가가 시장가에 근접한다 이게 뉴턴-랩슨의 핵심: 오차 ÷ 기울기(Vega) = 최적 조정량

실전 예시

콜 940, 시장가 12.35, 현물 933.25, 잔존 5일 [1회차] σ=30.00% → BS=10.64 → 오차 -1.71 → 조정 +4.03%p [2회차] σ=34.03% → BS=12.28 → 오차 -0.07 → 조정 +0.16%p [3회차] σ=34.19% → 오차 0.0003 → 수렴! IV = 34.19% 3회 만에 소수점 이하 4자리까지 정확한 IV를 찾았다

IV를 알면 뭐가 좋은가?

• 같은 행사가의 콜 vs 풋 IV 비교 → 수급 쏠림 파악 • 행사가별 IV 비교 → IV 스큐(Skew) 확인 • 어제 vs 오늘 IV 비교 → 변동성 방향 파악 • IV > HV → 시장이 과도하게 불안해하는 중 • IV < HV → 시장이 과도하게 안심하는 중 💡 증권사는 IV를 안 보여주거나 별도 화면에 숨긴다. 이 계산기의 존재 이유가 여기에 있다.

σ IV (Implied Volatility, 내재변동성)

IV란?

시장가격에 녹아있는 "기대 변동성" 옵션 가격 = f(현물, 행사가, 잔존일, 변동성, 금리) 다른 변수는 관측 가능하지만, 변동성만 미래의 값이다. → 시장가에서 역산한 변동성 = IV (내재변동성) → 과거 주가에서 계산한 변동성 = HV (역사적변동성)

IV가 높으면?

IV ↑ → 옵션 프리미엄 ↑ (콜·풋 모두) IV 낮을 때 매수 → 싸게 산다 IV 높을 때 매수 → 비싸게 산다 (IV 하락 시 손실)

IV 수준 판단 기준

KOSPI200 기준: • < 15%: 매우 낮음 — 시장 극도 안정 (옵션 매수 유리) • 15~25%: 정상 범위 • 25~35%: 높음 — 변동성 확대 구간 • 35~50%: 매우 높음 — 급등락, 매수 주의 • > 50%: 극단적 — 금융위기 수준 (VIX 공포지수 40+ 대응) S&P500 (SPX) 기준: • < 12%: 매우 낮음 • 12~18%: 정상 범위 • 18~25%: 높음 • 25~35%: 매우 높음 • > 35%: 극단적

어디서 확인하나?

방법 1: MTS에서 직접 일부 증권사 MTS는 행사가별 IV를 표시 (한국투자증권: 옵션시세 → IV 컬럼) 방법 2: 이 계산기의 IV 역산 탭 시장가를 넣으면 뉴턴-랩슨으로 IV를 역산 방법 3: 대표 지표 • KOSPI200: V-KOSPI200 (한국 VIX) • S&P500: VIX (CBOE 변동성지수)

핵심 한 줄

IV = 옵션의 "체감 물가". 높으면 비싸고, 낮으면 싸다. 방향(Delta)이 맞아도 IV 하락(Vega 손실)으로 돈을 잃을 수 있다. → 매수 전 반드시 IV 수준을 확인하라.

💰 시장가격이란?

시장가격 = MTS "현재가"

MTS 화면의 "현재가-콜" 또는 "현재가-풋" = 마지막 체결된 가격 = 시장에서 실제 거래되는 가격 예시: 콜 940 현재가 12.35 → 여기에 12.35를 입력

입력하면 뭘 비교하나?

이론가(BS) vs 시장가(MTS) 이론가 > 시장가 → 저평가 (매수 유리) 시장이 BS보다 싸게 팔고 있다 이론가 < 시장가 → 고평가 (매수 불리) 시장이 BS보다 비싸게 팔고 있다

왜 차이가 나는가?

BS 이론가는 하나의 IV로 수학적으로 계산한 "공정가격" 시장가에는 추가로 반영되는 것들: • 수급 — 매수자가 많으면 시장가 ↑ • 심리 — 공포/탐욕이 가격을 왜곡 • 호가 스프레드 — 매수/매도 호가 차이 • IV 스큐 — 행사가별 IV가 다름 괴리가 클수록 시장의 비효율 = 기회
💡 안 넣으면 이론가만 계산됩니다 (비교 생략)
